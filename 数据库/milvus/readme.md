官方文档：https://milvus.io/docs，提供的sdk包函常见的变成语言：
- python：核心sdk，使用较为方便
- java、go、nodejs，这些SDK 与 Milvus 服务器（尤其是 v2.x 系列）确实存在 API 不稳定或行为差异的情况
    - 无论是java、go，不同版本的sdk API接口常常不兼容
- 安装方式：pip 安装lite；docker compose启动standalone版本；k8s集群版本


核心概念：
- db，蕾丝RDMS的database
- collection，类似于RDMS的table
    - 一个collection可以有多个partition；一个collection可以有多个字段
- 标量类型
| 类型                             | 说明   |
| ------------------------------ | ---- |
| `Bool`                         | 布尔   |
| `Int8 / Int16 / Int32 / Int64` | 整数   |
| `Float`                        | 浮点   |
| `Double`                       | 双精度  |
| `VarChar`                      | 字符串  |
| `JSON`                         | JSON |
| `Array`                        | 数组   |

- 向量类型（Vector）
| 向量类型                  | 数据格式                   | 适合场景         | 查询性能    | 精确度        | 存储/内存占用 | 常见模型                            |
| --------------------- | ---------------------- | ------------ | ------- | ---------- | ------- | ------------------------------- |
| **FloatVector**       | 32-bit float           | 通用 AI 向量搜索   | 中       | **最高**     | 高       | OpenAI、BERT、SentenceTransformer |
| **Float16Vector**     | 16-bit float           | 大规模向量库       | 快       | 高（略低于FP32） | 中       | GPU embedding                   |
| **BFloat16Vector**    | 16-bit float（bfloat16） | AI训练/推理系统    | 快       | 高          | 中       | TPU / 深度学习框架                    |
| **BinaryVector**      | bit（二进制）               | 图像指纹 / hash  | **非常快** | 低          | **极低**  | 图像去重                            |
| **SparseFloatVector** | 稀疏向量                   | 关键词搜索 / 混合检索 | 中       | 高          | 低（稀疏存储） | BM25 / SPLADE                   |


- 索引
| 索引类型          | 核心原理             | 适合场景       | 数据规模建议    | 查询性能    | 构建速度 | 精确度（Recall）     | 内存占用         |
| ------------- | ---------------- | ---------- | --------- | ------- | ---- | --------------- | ------------ |
| **FLAT**      | 暴力遍历所有向量         | 小数据集、测试验证  | ≤100万     | 慢       | 非常快  | **100%（精确搜索）**  | 高            |
| **IVF_FLAT**  | 倒排文件（聚类分桶）+ 精确向量 | 通用向量检索     | 100万–1亿   | 中       | 快    | 高（≈90–95%）      | 中            |
| **IVF_SQ8**   | IVF + 标量量化压缩     | 内存有限场景     | 100万–1亿   | 快       | 中    | 中高（≈85–92%）     | 低            |
| **IVF_PQ**    | IVF + 产品量化       | 超大规模向量库    | 1000万–10亿 | 快       | 慢    | 中（≈75–90%）      | **非常低**      |
| **HNSW**      | 分层小世界图（图搜索）      | 高性能实时搜索    | ≤1亿       | **非常快** | 慢    | **很高（≈95–99%）** | 高            |
| **DISKANN**   | 图结构 + 磁盘索引       | TB级数据、内存不足 | 1亿–10亿+   | 快       | 慢    | 高               | **低内存，依赖磁盘** |
| **AUTOINDEX** | Milvus 自动选择索引    | 不想调参       | 任意        | 自动优化    | 自动   | 自动              | 自动           |


- 距离类型：向量相似度计算方式：
| 类型        | 说明    |
| --------- | ----- |
| `L2`      | 欧式距离  |
| `IP`      | 内积    |
| `COSINE`  | 余弦相似度 |
| `HAMMING` | 二进制距离 |
| `JACCARD` | 集合相似度 |
