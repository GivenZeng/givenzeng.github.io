milvus核心功能是对向量数据的增删改查。如何根据媒体数据（文本、音频、视频等）生成向量，用户可以自主考虑。


# 向量生成方式和对比
下面是 **三种常见 Python 文本生成向量方案**的对比：

## 使用 OpenAI 生成向量
pip install openai
```pythone
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY"
)

text = "Milvus is a vector database"

resp = client.embeddings.create(
    model="text-embedding-3-small",
    input=text
)

vector = resp.data[0].embedding

print("vector dim:", len(vector))
print(vector[:10])  # 打印前10个
```

## 使用本地模型（sentence-transformers）
```python
# pip install sentence-transformers
from sentence_transformers import SentenceTransformer

model = SentenceTransformer("all-MiniLM-L6-v2")

texts = [
    "Milvus is a vector database",
    "Python is a programming language"
]

vectors = model.encode(texts)

print("dim:", len(vectors[0]))
print(vectors[0][:10])
```

## 用 Milvus 自带 embedding
```python
from pymilvus import model

embedding_fn = model.dense.SentenceTransformerEmbeddingFunction(
    model_name='all-MiniLM-L6-v2'
)

docs = [
    "Milvus is a vector database",
    "Vector search is powerful"
]

vectors = embedding_fn.encode_documents(docs)

print(len(vectors[0]))
```

# 长文本怎么划分
不要直接 embedding 整个文档。必须 chunk。推荐 chunk 大小。最佳实践：300 – 500 tokens

| chunk 太大 | chunk 太小 |
| -------- | -------- |
| 召回不精准    | 上下文不足    |

比较合理的流程：
1. 将原文本保存到DB存储：得到原文id、原文
2. 将原文拆分成多个段落（chunk）：chunk_size = 400 tokens，chunk_overlap = 50 tokens
3. 使用向量生成工具根据这些段落生成向量，写入milvus：段落id（可使用milvus自增生成）、段落向量、段落原文、原文id，向量索引如下
```
index_type: HNSW
metric_type: COSINE
```

查询流程
1. 根据用户输入的问题生成向量 query_vector
2. 使用query_vector调用milvus查询相关性较强的m个段落
3. 【推荐】对这n个段落进行rerank，从rerank结果得到top k个段落。（可以使用BGE Reranker（BAAI General Embedding Reranker）是一种由北京智源人工智能研究院（BAAI）开发的开源重排序模型，用于信息检索、问答与语义搜索中的候选结果重新排序。）
4. 将top k个段落的原文、用户输入的问题等作为上下文输入提供给LLM


# http server
python代码，用于生成向量。使用flask框架，生成一个http server，提供一个API：输入参数为text，返回这个text对应的词向量，response为json格式
```python
from flask import Flask, request, jsonify
from sentence_transformers import SentenceTransformer

# 初始化模型（服务启动时加载一次）
model = SentenceTransformer("all-MiniLM-L6-v2")

app = Flask(__name__)
@app.route("/embedding", methods=["POST"])
def embedding():
    data = request.get_json()

    if not data or "text" not in data:
        return jsonify({
            "error": "missing 'text' field"
        }), 400

    text = data["text"]

    # 生成向量
    vector = model.encode(text)

    return jsonify({
        "text": text,
        "dim": len(vector),
        "vector": vector.tolist()
    })


if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000)
```
